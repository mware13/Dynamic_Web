About This Project

This website is a university project developed with the intention of demonstrating my knowledge of UI and UX design for web forms, modern web standards, Javascript objects, JSON and AJAX, as well as server-side programming. In addition to the technical demonstration, the CommuniTypography project is intended to be a community-driven and user-generated typography experiment.
